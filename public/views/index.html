Welcome to the States API

Explore various endpoints of the API to learn more about U.S. states!

Endpoints

  • /states: View all states
  • /states?contig=true: View contiguous states
  • /states?contig=false: View non-contiguous states (AK, HI)
  • /states/:state: View details for Kansas (replace `KS` for other states)
