Welcome to the States API

A Node.js REST API project for exploring U.S. states data.

About the Project

The States API allows users to interact with data on all 50 U.S. states, including capitals, populations, admission dates, and fun facts. Use the endpoints below to explore our API.

Available Endpoints

  • /states: View all state data
  • /states?contig=true: View contiguous states (excludes AK and HI)
  • /states?contig=false: View non-contiguous states (AK and HI)
  • /states/:state: View details for Kansas (replace `KS` with another state)
  • /states/:state/funfact: View a random fun fact about Kansas (replace `KS` with another state)
  • /states/:state/capital: View the capital of Kansas (replace `KS`)
  • /states/:state/nickname: View the nickname of Kansas (replace `KS`)
  • /states/:state/population: View the population of Kansas (replace `KS`)
  • /states/:state/admission: View Kansas's admission date (replace `KS`)
